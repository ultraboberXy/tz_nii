# tz_nii
for smo
Возможности:
Загрузка списка доступных дат с сервера RIA
Получение точек (событий) за выбранную дату
Сохранение данных в локальную SQLite БД
Табличное представление с сортировкой по всем колонкам
Выбор даты через выпадающий список
Очистка локальной базы данных


Язык: C++17
Фреймворк: Qt 6.11.1 (Widgets, Network, Sql)
Сборка: CMake, MinGW 64-bit


Программа разрабатывалась в программе Qt Creator 19, в случае некорректной работы жду отзыв.
